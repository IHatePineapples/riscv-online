RISC-V Online

RISC-V Assembly Web Interpreter written in C++ for the Raspberry Pi Pico W.

Building from source

Requirements

  • CMake
  • Make
  • GCC/G++ or equivalent (supports c++17)
  • Raspberry Pi Pico W
  • Boost (Header-only)

Configure:

cmake .

Build:

make -j$(nproc)

Testing:

cd src/test
cmake .
make
make test
